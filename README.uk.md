# Проєкт інтернаціоналізації контенту спільноти XMCL

Цей репозиторій містить інтернаціоналізацію (i18n) для контенту спільноти, такого як моди, пакети ресурсів та шейдери для XMCL.

[English README тут](./README.en.md) [Русский README здесь](./README.ru.md) [Deutsches README hier](./README.de.md) [Polskie README tutaj](./README.pl.md) [简体中文 README](./README.md)

## Навіщо потрібен цей репозиторій?

Оскільки багатьом користувачам звичніше бачити локалізовані назви, а не оригінальні англійські. Мета цього репозиторію — допомогти більшій кількості користувачів краще використовувати XMCL і навіть надати орієнтир для інших розробників.

## Як зробити свій внесок

Ми використовуємо git для керування перекладами. Ви можете зробити форк цього репозиторію та надіслати pull request зі своїми перекладами.

### Як додати переклад для моду/пакета ресурсів/шейдера?

Дані перекладів для модів, пакетів ресурсів та шейдерів зберігаються відповідно у `<locale>/mods.csv`, `<locale>/resourcepacks.csv` та `<locale>/shaderpacks.csv`.

`<locale>` — це код локалі для мови, яку ви хочете перекласти. Наприклад, `zh_cn` — це код локалі для китайської мови.

Ви можете додати новий рядок у відповідний файл, щоб додати новий переклад. Стовпці розділяються комою `,`.

Рядок має відповідати наступному формату:

```
<name>,<modrinthId>,<curseforgeId>,<description>
```

`name` — це локалізована назва моду/пакета ресурсів/шейдера. `description` — це локалізований опис моду/пакета ресурсів/шейдера.

Ці два значення відображатимуться в XMCL.

Якщо ви не знаєте `modrinthId` або `curseforgeId`, ви можете залишити поле порожнім, але необхідно вказати хоча б один з ідентифікаторів `modrinthId` або `curseforgeId`.
